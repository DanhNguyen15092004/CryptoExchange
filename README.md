# Crypto Chart Client

A simple React application with TailwindCSS and Lightweight Charts to display cryptocurrency price charts.

## Installation

```bash
cd client
npm install
```

## Run Development Server

```bash
npm run dev
```

The app will run on http://localhost:3000

## Features

- Real-time crypto price charts using Lightweight Charts
- Beautiful UI with TailwindCSS
- Switch between BTC, ETH, and SOL
- Responsive design
- Dark theme

## Tech Stack

- React 18
- Vite
- TailwindCSS
- Lightweight Charts
